# 帳款管理系統

多人共用的應收／應付帳款管理系統，資料存放於 Google Sheets。

## 使用方式

直接用瀏覽器開啟本專案的 GitHub Pages 網址即可，無需安裝任何軟體。

## 設定步驟（第一次）

1. 打開 `index.html`，找到第一行 `const API_URL = ...`
2. 填入你的 Google Apps Script 部署網址
3. 上傳到此 GitHub Repository
4. 開啟 GitHub Pages 後，將網址分享給同事

## 注意事項

- 需要網路連線（資料存在 Google Sheets）
- 畫面每 30 秒自動刷新一次
- 點「立即更新」可手動同步最新資料
